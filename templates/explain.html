Scraper Instructions

Run the Scraper Functions Before Filtering! You can click any of the buttons on that page to scrape the webpage or twitter handle. You can even input any twitter username to scrape those results!

Filter Instructions

Run the Filter Functions After Scraping! You can choose whether to filter a twitter account, or a web page. You also have the option to choose whether to scrape by groups or not (only for CSK & KKM)

AD

Data Storage Instructions

Under the project folder on your computer, each scrape/filter will put a file with new information in a folder labelled as the current date (YYMMDD), and each scrape will also append any new info to the allTime folder

How To Set Up

1. Put project in Desktop

2. Click FirstTime.sh

3. Click ClickHere.sh

4. Click ClickHere2.sh

5. Any time you would like to run it in the future, just follow steps 3 & 4.
